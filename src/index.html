INFINITAS Table Maker 도움말

1) 이 앱이 하는 일

INFINITAS 플레이 기록(tracker.tsv)을 기반으로 SP10/11/12 HARD 서열표를 개인화하고, 히스토리/목표 관리까지 한 번에 하는 데스크톱 도구입니다.

2) 최초 사용 순서

  1. 앱 실행 후 계정 생성: DJ NAME, INFINITAS ID, 아이콘을 등록합니다.
  2. 데이터 불러오기(tsv) 또는 갱신으로 데이터를 가져옵니다.
  3. 서열표에서 현재 상태를 확인하고, 목표 탭에서 목표를 등록합니다.

3) 갱신(Reflux 연동) 사용법

  1. 갱신 버튼을 누르면 Reflux 최신 버전 확인/갱신 후 콘솔이 실행됩니다.
  2. 이 상태로 INFINITAS를 플레이합니다.
  3. 게임 종료 후 tracker.tsv 반영이 확인되면 갱신 완료 버튼이 활성화됩니다.
  4. 갱신 완료를 눌러 팝업을 닫으면 데이터/히스토리가 업데이트됩니다.
  • 갱신 취소를 누르면 Reflux 프로세스와 콘솔이 종료됩니다.
  • 갱신 중 목표 카드(설정 ON 시)는 진행중 목표만 표시합니다.
  • 플레이 중 달성된 목표 카드는 카드 색상이 달성 상태로 바뀝니다.

4) 서열표 페이지

  • SP10 HARD / SP11 HARD / SP12 HARD 테이블 전환
  • 일반 / WIDE 보기 전환
  • 램프순 정렬 / 이름순 정렬
  • 다운로드: 현재 테이블을 이미지로 저장
  • 곡 클릭: 램프, 스코어, MISS, RATE 등 상세 팝업
  • 미해금 곡은 어둡게 표시되며 정렬 시 후순위로 처리됩니다.

5) 히스토리 페이지

  • 왼쪽: 업로드/갱신 시점 목록, 오른쪽: 해당 시점 상세
  • 카테고리: 램프 갱신 / 스코어 갱신 / 목표 갱신
  • 최초 기록은 (최초), 최신 기록은 (최신)으로 표시
  • 롤백: 선택 시점 이후 히스토리 삭제 + 해당 시점 상태로 복원

6) 목표 페이지

  • 목표 종류: CLEAR / SCORE / RANK
  • 곡 검색/채보 선택 후 목표 추가/갱신
  • 달성/진행 상태가 자동 갱신되며 카드 배경색으로 구분됩니다.
  • 현재 목표 내보내기, 목표 불러오기, 달성 목표 전체 삭제, 목표 전체 삭제 지원

7) 계정/데이터 저장 위치

계정/히스토리/목표 데이터는 Windows 사용자 프로필의 앱 데이터 경로에 저장됩니다. (예: %APPDATA%/infinitas-rank-table-maker)

8) 권한 관련 안내

  • 이 앱은 메모리 접근이 필요한 Reflux 연동을 위해 관리자 권한이 필요합니다.
  • 앱 실행 시 자동으로 관리자 권한 재실행을 시도하며, 거부하면 실행을 종료합니다.

9) 문제 해결

  • 갱신 완료가 안 뜰 때: 게임 플레이 후 게임 종료까지 진행했는지 확인
  • 데이터가 이상할 때: 히스토리 롤백으로 이전 시점 복구 후 다시 갱신
  • 표시가 어긋날 때: 앱 재실행 후 서열표 새로고침 실행
  • Google 연동 전: INFINITAS ID를 정확히 입력하세요. 오입력 시 다른 계정과 충돌해 연동이 거부될 수 있습니다.
